WidgetVersion

@get:Composable
@get:ReadOnlyComposable
val WidgetVersion: UInt

Exposes the version of the widgets for conditional logic (such as progressive enhancement).

If you have Button introduced in version 1 and FancyButton introduced in version 3 but your minimum-supported version is 2, users of FancyButton will need to use a conditional around FancyButton to fallback to Button.

if (WidgetVersion >= 3) {
  FancyButton("text")
} else {
  Button("text")
}
