Uncertain

Uncertain(0b000)
Content copied to clipboard

Indicates that nothing is certain about the current state of the parameter. It could be different than it was during the last execution, or it could be the same, but it is not known so the current function looking at it must call equals on it in order to find out. This is the only state that can cause the function to spend slot table space in order to look at it.
