Cell

class Cell<C : Any, W : Any>

This class keeps track of whether a cell is bound to an on-screen UI.

While it's bound to the UI, its contents can change:

  • A placeholder can be swapped for a loaded widget, if the loaded window moves to include this cell.

  • Symmetrically, a loaded widget can be swapped for a placeholder, if the loaded window moves to exclude this cell.

If it currently holds a placeholder, this placeholder must be released to the processor's placeholder queue when it is no longer needed. This will occur if cell is reused (due to view recycling), or because it is discarded (due to the table discarding it). This class assumes that a cell that is discarded will never be bound again.
